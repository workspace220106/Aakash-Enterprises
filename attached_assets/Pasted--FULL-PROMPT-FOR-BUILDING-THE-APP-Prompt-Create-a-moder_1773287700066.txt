🚀 FULL PROMPT FOR BUILDING THE APP

Prompt:

Create a modern, animated inventory and sales management web app for a Cold Drinks Wholesaler Shop.

The application must help a wholesaler track sales, manage stock, record customers, and analyze monthly performance.

The UI must be colorful, animated, simple to use, and optimized for quick billing in a shop environment.

🧊 App Name

CoolTrack – Cold Drinks Sales Manager

Design the interface to be:

• Very colorful and vibrant (soft drink theme)
• Smooth micro-animations and transitions
• Glassmorphism + modern dashboard UI
• Easy for fast billing in a busy shop
